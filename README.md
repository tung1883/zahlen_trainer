# Zahlen Hörverstehen Trainer
Gestern war mein letzter Tag an der HANU-Universität, aber Hörverstehen bleibt für mich ein großes Problem – besonders bei Zahlen, Daten usw.  
Ich konnte keine gute Ressource finden, die genau diese Art von Übungen fokussiert. Also habe ich mir gedacht: *Warum mache ich es nicht einfach selbst?*

Und so ist dieses kleine Projekt entstanden. Hoffentlich kann es auch euch beim Deutschlernen helfen.

Für die Beschreibung der Funktionen benutze ich jetzt Englisch, damit ich nicht noch mehr ChatGPT brauche

---
## Features
- Multiple training modes:
  - Numbers
  - Prices
  - Dates (with German month names)
  - Phone numbers (chunked format)
  - Addresses
- Level system:
  - Level 1: 1–99
  - Level 2: 100–999
  - Level 3: 1000+
- Game mode:
  - Automatic rounds
  - Time limit (gets faster over time)
  - 3 lives system
- Input improvements:
  - Auto-check while typing
  - Optional manual check
  - Input length hints (placeholder)
- Audio:
  - German speech using browser TTS
  - Replay button
